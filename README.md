# SPO-Student-Portal-June-2026
Hi, so this is a very basic portal built using HTML, Go and PostgreSQL. 
As of now, logins and signups are perfectly handled.
A  student can submit verification requests to the existing domains.
The master admin can create and remove domain admins and make their own domain names. 
The rest of the buttons just lead to a placeholder page.
Anyone who tries to access the homepages directly will not be allowed (didn't have enough time to implement this for the pages beyond the homepage though)
There isn't any password hashing or salting yet.
